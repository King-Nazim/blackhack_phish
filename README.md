
## Installation :

* `apt update`
* `apt install git -y`
* `git clone https://github.com/King-Nazim/blackhack_phish`
* `cd nexphisher`
#### > SETUP : `bash setup`
#### > SETUP [TERMUX] : `bash tmux_setup`
#### > Run : `bash phish`
